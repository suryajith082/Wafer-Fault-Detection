About This

While training data should be stored on AWS S3 Bucket

Bucket name should be mentioned on input. To predict select Home from navigation bar.

Project is not for public. It uses AWS S3 Bucket service for getting files and storing prediction output

Program logs are stored on MongoDB Atlas

On the basis of DSA agreement, file naming conventions and their properties are stored on MongoDB

While training it will check all those naming conventions and properties of all files, on the basis of that bad files will be classified and names of that file are mailed to client.

The best suitable ML model is selected or binary file generation is done by its own. No middle person is needed

Preprocessing data transformation and model saving all those kind of steps are done itself.
